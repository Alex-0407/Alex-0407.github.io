flex-direction

flex-direction 顺序指定了弹性子元素在父容器中的位置。

语法

flex-direction: row | row-reverse | column | column-reverse

row：横向从左到右排列（左对齐），默认的排列方式。

flex item 1
flex item 2
flex item 3

row-reverse：反转横向排列（右对齐，从后往前排，最后一项排在最前面。

flex item 1
flex item 2
flex item 3

column：column：纵向排列。

flex item 1
flex item 2
flex item 3

column-reverse：反转纵向排列，从后往前排，最后一项排在最上面。

flex item 1
flex item 2
flex item 3
