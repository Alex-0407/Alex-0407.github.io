flex-wrap

flex-wrap 属性用于指定弹性盒子的子元素换行方式。

语法

flex-flow: nowrap | wrap | wrap-reverse

nowrap - 默认， 弹性容器为单行。该情况下弹性子项可能会溢出容器。

flex item 1
flex item 2
flex item 3

wrap - 弹性容器为多行。该情况下弹性子项溢出的部分会被放置到新行，子项内部会发生断行。

flex item 1
flex item 2
flex item 3

wrap-reverse -反转 wrap 排列。

flex item 1
flex item 2
flex item 3
